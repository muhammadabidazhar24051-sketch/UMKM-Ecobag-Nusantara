##  Fitur Utama Website

### 1. Halaman Beranda (`index.html`)
- Menampilkan pengenalan singkat UMKM.
- Banner promosi atau pengantar singkat.

### 2. Halaman Produk (`produk.html`)
- Daftar produk seperti totebag, sling bag, dompet, dan lainnya.
- Setiap produk dilengkapi gambar, nama, dan deskripsi singkat.

### 3. Halaman Galeri (`galeri.html`)
- Menampilkan foto-foto kegiatan pembuatan ecobag, pameran, dan dokumentasi UMKM.

### 4. Halaman Kontak (`Kontak.html`)
- Informasi kontak yang dapat dihubungi.
- Tautan ke sosial media UMKM.

### 5. Halaman Webinar (`Webinar.html`)
- Informasi mengenai acara webinar atau event yang berkaitan dengan UMKM.

### 6. Audio Pengantar
- File `AudioEcoBag.mp3` sebagai audio tambahan pada website.
